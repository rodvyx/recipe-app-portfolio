# RecipeHub

Modern recipe discovery web application built with JavaScript.

## Features

- Search recipes
- View recipe details
- Adjust servings
- Bookmark favorites
- Recently viewed recipes
- Upload your own recipes
- Responsive modern UI
- LocalStorage persistence

## Tech Stack

- JavaScript (ES6+)
- MVC Architecture
- Parcel
- Sass / SCSS
- REST API Integration

## Purpose

Originally based on an older learning project, fully redesigned and upgraded into a portfolio-ready application.

npm install
