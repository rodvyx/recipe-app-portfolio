# RecipeHub

Modern recipe discovery web application built for portfolio presentation.

## Live Features

- Search recipes by keyword
- View detailed recipe instructions
- Adjust servings dynamically
- Bookmark favorite recipes
- Recently viewed recipes
- Upload your own custom recipes
- Responsive modern UI
- Persistent local storage data

## Tech Stack

- JavaScript (ES6+)
- MVC Architecture
- Parcel Bundler
- Sass / SCSS
- REST API Integration
- LocalStorage

## Project Purpose

This project began as an older learning application and was fully refactored into a modern production-style portfolio project with upgraded UI, better UX, cleaner structure, and additional features.

## Key Improvements Made

- Full rebrand to RecipeHub
- Modern visual redesign
- Recently viewed system
- Improved search UX
- Cleaner upload modal
- Better component structure
- Cleaner helper logic
- Portfolio-ready presentation
